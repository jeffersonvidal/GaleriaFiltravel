# Galeria Filtrável
Modelo de galeria de imagens feito com HTML, CSS e JavaScript.

Ideal para apresentação de portfólios de sites pessoais ou empresariais.

## TECNOLOGIAS UTILIZADAS
- HTML 5
- JavaScript
- CSS 3
